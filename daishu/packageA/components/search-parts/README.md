#图片组件 (img-parts)

## 用法

```code

<img-parts url="/images/1.jpg" width="200" height="200" />

```

## 参数

| 参数名 | 是否必填 | 类型 | 说明 |
|--|--|--|
| url | 是  | String| 图片路径 |
| width | 否  | Number |  宽度，默认为空，即最大填充 |
| height | 否  | Number |  高度，默认为空，即图片比例高度 |
| isbackground  | 否 | Boolean |  是否为背景， 如果设置true, width和height值必填 |
| errorUrl | 否  | Stirng |  找不到图片是，使用该图片占位 |
